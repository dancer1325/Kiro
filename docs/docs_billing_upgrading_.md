# Upgrading your subscription - IDE - Docs - Kiro

Source: https://kiro.dev/docs/billing/upgrading/

---

Upgrading your Kiro plan gives you access to more features and higher credit limits. Once you upgrade your subscription, it'll take effect immediately. You'll be charged a prorated fee, but immediately get access to your new plan's full credit limits.
Upgrading a social or AWS Builder ID account for the first time
Use the following instructions to upgrade a social account or an AWS Builder ID account for the first time in Kiro.
InfoCurrently, you can only upgrade an AWS Builder ID account if it is not linked to an AWS account. If you access Kiro through a social login provider or a Builder ID account, your payments are currently processed by Stripe and Amazon Web Services, Inc. is your AWS Contracting Party under your agreement with us.
In Kiro, select the profile icon.
Choose Upgrade Plan.
Select which plan you'd like to upgrade to.
Choose Open. You will be redirected to your default web browser to complete upgrade process.
In the browser, confirm the details of the plan and then enter your email.
Select your payment method, enter your payment details, and then choose Subscribe.
After confirming your subscription, you can select Get started to return back to the Kiro IDE.
Upgrading a social or AWS Builder ID account
After you've upgraded plans for the first time, you will no longer see the plan options in the Kiro IDE. Instead, use the following instructions to upgrade a social account or an AWS Builder ID account.
In Kiro, select the profile icon.
Choose Manage Plan. You will be redirected to your default web browser to complete the upgrade process.
In the browser, choose Update subscription, select which plan you'd like to upgrade to, and then choose Continue.
Confirm the details of the plan and then choose Confirm.
After confirming your subscription, you can select Return to Kiro to return to the Kiro IDE.
