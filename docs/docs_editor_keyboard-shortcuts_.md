Custom shortcuts
You can customize keyboard shortcuts in Kiro by using the following instructions:
Open the Command Palette (Cmd+Shift+P / Ctrl+Shift+P).
Search for Keyboard Shortcuts.
Select Preferences: Open Keyboard Shortcuts.
Select the command that you want to modify.
Choose the pencil icon and enter your preferred shortcut.
